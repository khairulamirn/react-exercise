props (JavaScript object) = read-only properties that are shared between components
        A parent component can send data to a child component
        <Component key=value />

propTypes = a mechanism that ensures that the passed value is of the correct datatype. age: PropType.number debugging purposes

defaultProps = default values for props in case they are not passed from the parent component name: "Guest"

conditional rendering = allows you to control what gets rendered in your application based on certain conditions (show, hide, or change components)
